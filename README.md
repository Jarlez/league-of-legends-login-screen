## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- JavaScript

## 💻 Projeto

Clone da tela de login do jogo 'League of Legends' usando HTML, CSS e JavaScript. O foco principal foi aprimorar um formulário completo e implementar um modal em uma página estática.
